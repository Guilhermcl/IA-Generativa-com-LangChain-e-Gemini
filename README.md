# IA-Generativa-com-LangChain-e-Integra-o-Gemini
Este repositório contém o código de um sistema de IA generativa desenvolvido utilizando a biblioteca LangChain, permitindo a integração fluida de múltiplos Modelos de Linguagem (LLMs), como o Google Gemini, GPT e Claude, com o objetivo de criar uma plataforma modular, escalável e flexível.
